# 狐狸躺 Steam Logo 面板 — 多色列印說明

黑面板 ＋ 平面 Steam logo（藍圓＋白外環＋白活塞）＋ 132mm 狐狸凸起躺在上面。
六個零件都在**同一座標對齊**，AMS 4 色。`reference.png` 是完成後的樣子。

| 檔案 | 內容 | 高度 | 建議線材 |
|---|---|---|---|
| `1_panel.stl` | 黑面板（正面挖淺圓槽放 logo） | — | 黑 |
| `2_blue_circle.stl` | 藍圓（平、齊面，活塞處已挖空） | 齊面 1mm | Steam 藍 |
| `3_white_logo.stl` | 白活塞＋白外環（平、齊面） | 齊面 1mm | 白 |
| `4_fox_orange.stl` | 狐狸橘身體 | 凸起 1.5mm | 橘 |
| `5_fox_white.stl` | 狐狸白肚子 | 凸起 1.5mm | 白 |
| `6_fox_black.stl` | 狐狸黑輪廓＋五官 | 凸起 1.5mm | 黑 |

## 層次概念
- **藍圓 / 白 logo 是「平的、齊面」**，填在面板正面的淺圓槽裡，和黑面板同高。
- **狐狸「凸起 1.5mm」趴在上面**；狐狸蓋到的地方，logo 就藏在底下 → 「狐狸躺在 logo 上」。
- 白活塞(3) 與狐狸白肚子(5) 都用白線材（高度不同，正常）。

## 切片步驟（Bambu Studio / OrcaSlicer / PrusaSlicer 通用）
1. **六個 STL 一起匯入**；跳出「合併為單一物件的多個零件 / load as parts」選【是】，保持對齊。若跑位就全選 → 右鍵「**組裝 Assemble**」。
2. **指定線材**：4 色（黑 / 藍 / 白 / 橘）。零件 3 與 5 指定**同一個白**。
3. **擺放方向**：**狐狸那面朝上**（凸起層在上，色彩最漂亮、免支撐）。
4. 開 AMS／多色，切片、列印。

## 尺寸
- 面板 156 × 126 × 14.5 mm
- 藍圓 Ø115、白外環到 Ø121、狐狸寬 132mm、凸起 1.5mm

## 注意
- 需要 **4 槽 AMS**（或多材料）才能一次印四色。
- 狐狸四肢有些垂到圓外／黑底上，是刻意的「大剌剌躺上去」效果。
- 藍色目前 `#1b3a6b`（Steam 深藍），外環 3mm —— 想改再跟我說，都是改一個值重出。
