# 狐狸多色浮雕面板 — 列印說明

四個零件都在**同一座標系對齊**，載入切片軟體後保持原位置即可自動對齊。

| 檔案 | 內容 | 建議線材 |
|---|---|---|
| `1_panel.stl` | 面板本體（原始檔，未改動） | 黑色（或你喜歡的顏色） |
| `2_fox_orange.stl` | 狐狸橘色身體 | 橘色 |
| `3_fox_white.stl` | 白色肚子／臉 | 白／米白 |
| `4_fox_black.stl` | 黑色輪廓＋眼睛＋鼻子＋腳掌 | 黑色 |

## 切片步驟（Bambu Studio / OrcaSlicer / PrusaSlicer 通用）

1. **一次選取四個 STL 一起匯入**。若跳出「是否合併為一個物件的多個零件（load as parts / single object）」→ 選【是】，這樣會保持對齊。
   - 若是各自匯入而跑掉位置：全選 → 右鍵 →「**組裝 / Assemble**」把它們對回原位。
2. **指定線材**：每個零件依上表指定顏色。
3. **擺放方向**：把**狐狸那一面朝上**（狐狸凸起 1.5mm，朝上印色彩層最漂亮、不需支撐）。
4. 開 AMS／多色，切片、列印。

## 尺寸
- 面板 156 × 126 × 14.5 mm
- 狐狸寬 115mm、置中、凸起 1.5mm（三色同高平貼）

## 注意
- 黑色輪廓在**黑色面板**上會融進背景 → 外框看不出來，狐狸靠橘／白呈現，黑色只在橘色上的眼睛、腳掌看得到。
- 想讓**外框也明顯** → 面板改挑非黑色（例如深灰、白、木色）。
- 三色邊界是同一像素分色描出來的，密合無縫；相鄰零件在切片時共用邊界牆，正常列印即可融合。
